Web based live chat using Angular 7 and Firebase/Firestore

User authentication provided by Firebase for secure user login and registration

Deployed at https://tuffypub.firebaseapp.com/

To install dependencies, run npm install on virality directory, run npm install on functions directory.

ng build in virality directory, then ng serve to view locally.
